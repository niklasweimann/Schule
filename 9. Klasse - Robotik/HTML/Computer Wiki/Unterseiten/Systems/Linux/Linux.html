Freie, unix-ähnliche Mehrbenutzer-Betriebssysteme basierend auf dem Linux-Kernel und wesentlich auf GNU-Software. Die kommerzielle Verbreitung wurde 1992 durch die Lizenzierung des Linux-Kernels unter der freien Lizenz GPL ermöglicht.

Linux ist ein modular aufgebautes Betriebssystem und wird von Softwareentwicklern der ganzen Welt, welche an verschiedenen Projekten mitarbeiten, weiterentwickelt. Unternehmen, Non-Profit-Organisationen und viele Freiwillige sind daran beteiligt.
Wenn Linux an Computern zum Einsatz kommt, werden meist Linux-Distributoren verwendet, deren Aufgabe es ist den Linux-Kernel mit verschiedener Software zu einem Betriebssystem zusammen zu fassen. Dieses ist dann für die Endnutzung geeignet.
Dabei passen viele Distributoren und versierte Benutzer den Kernel an ihre eigenen Zwecke an.

Linux wird beispielsweise verwendet auf:

  • Desktop Rechnern
  • Servern
  • Mobiltelefonen
  • Routern
  • Netbooks
  • Embedded Systems
  • Multimedia-Endgeräten
  • Supercomputern
Im Server-Markt und im moblien Bereich ist Linux eine feste Größe. Im Desktop-Bereich jedoch eine noch geringe, aber stetig wachsende Rolle hat.
