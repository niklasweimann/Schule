Windows war ursprünglich eine grafische Erweiterung von MS-DOS. Windows XP war das erste Betriebssystem von Microsoft, welches nicht mehr auf DOS basierte. Der Name "Windows" beruht auf dem Design, welches eben wie ein Fenster (engl. "Windows") aussieht. Windows ist vorallem auf Computern und Handys verbreitet. Aber auch die meisten Kassensysteme nutzen Windows. Die Windows-Betriebssysteme haben einen geschätzten Marktanteil von 82 Prozent.
