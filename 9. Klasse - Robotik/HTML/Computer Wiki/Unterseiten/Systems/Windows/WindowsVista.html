Windows Vista ist sowohl als 32-bit Version als auch als 64-bit Version erhältlich. Die K-Versionen ohne Windows Media Player und Windows Messenger sind nur in Korea erhältlich.
Die N-Versionen ohne Windows Media Player dagegen sind ausschließlich in der Europäischen Union erhältlich.

Windows Vista wurde insgesamt in zehn Versionen verkauft, wovon die Starter-Variante nicht in Europa veröffentlicht wurde:

  • Windows Vista Starter
  • Windows Vista Home Basic
  • Windows Vista Home Basic N
  • Windows Vista Home Basic K
  • Windows Vista Home Premium
  • Windows Vista Business
  • Windows Vista Business N
  • Windows Vista Business K
  • Windows Vista Enterprise
  • Windows Vista Ultimate
  • Windows Embedded, eine modular aufgebaute Version von Windows Vista für Eingebettete Systeme



Windows Vista ist der direkte Nachfolger von Windows XP und macht mit seinen Features und dem schicken Design einiges her. Jedoch ist die Arbeitsgeschwindigkeit von Vista sehr schlecht, mit ein Grund warum Microsoft mit Vista nicht an den Erfolg von Windows Xp anschließen konnte.

Neues in Windows Vista

  1. Aero-Oberfläche, Sidebar mit Minianwendungen und visuelle und transparente Effekte verleien Vista im vergleich zu Windows Xp ein neues Schickes Design.
  2. Die Systemsteuerung wurde in der Bedienung und dem Design angepasst.
  3. Startmenü ist nun kompakter gestalltet, mit einer Liste, in der die Programme angezeigt werden und nicht wie bei Windows Xp mit einem Menü, dass fast über den ganzen Bildschirm reicht.
  4. Das ausführen von Verwaltungsprogrammen erfodert Administratoren rechte und muss extra bestätigt werden.
  5. Zudem ist nun ein Jugendschutz erstmals direkt im System intregriert.
  6. Erneuerung für den Windows Explorer.
  7. Outlook express wird zu Windows Mail.
  8. Einführung der Windows Fotogalerie und dem Vista Media Center.
