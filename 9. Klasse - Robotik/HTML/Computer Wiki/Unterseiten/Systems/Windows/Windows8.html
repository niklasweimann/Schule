Windows 8 unterstützt nicht nur die herkömmliche x86-, sondern auch die ARM-Architektur der drei größten Produzenten Nvidia, Qualcom und Texas Instruments.

Windows 8 ist in vier Versionen erschienen:

  • Windows 8
  • Windows 8 Pro
  • Windows 8 Enterprise
  • Windows RT (Version für ARM-Prozessoren)

Neues in Windows 8

  1. Bei Windows 8 kam zum ersten Mal die neue Metro Oberfläche zum Einsatz, jedoch kam dieses Konzept bei den Usern von Desktop Computern nicht ganz so gut an wie der Vorgänger Windows 7, da sich die Metro Oberfläche mit der Maus eher schlecht bedienen läst.
  2. Der Windows Explorer wurde an das Design von Windows Office angepasst.
  3. In den Ecken verstcken sich nun Funktionen, wie z.B. dass wechseln zwischen laufenden Programmen oder der direkt Zugriff auf die Systemsteuerung.
  4. Zur vollständigen Nutzung des PCs wird ein Windows-Live-Account benötig.t
  5. Sky-Drive wurde fest in das System integriert.
  6. Programme bzw. jetzt Apps wie der Internet-Explorer werden nun im Vollbild-Modus angezeigt.
  7. Task-Manager wurde geupdatet.
  8. Synchronisation der Benutzereinstellungen ist nun möglich, allerdings nur mit einem Live-Account.
