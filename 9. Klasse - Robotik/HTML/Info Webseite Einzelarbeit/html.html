Grundbefehle
<html> </html> Öffnet und schließt das Html Dokument und steht oft in der Ersten und in der Letzten Zeile einer Htmlseite und ist somit der Grundbefehl, den jeder kenne sollte
<head> </head> Im "Header" der Seite befinden sich die Meta-Tags und verweise auf Css-Seiten (Dateine, in dem der Aufbau der Seite via Script beschrieben wird).
<body> </body> Im "Body" der Seite befindet sich die Eigentliche Seite, also Texte, Bilder, Videos oder Ähnliches. Es ist somit der Teil, indem der Inhalt steht.

Textformatierung
<p> </p> Dieser Befehl ermöglicht es dir einen Text einzugrensen und somit einen neuen Absatz zu erstellen.

  • <Da ist eine Zahl,

Verlinkung von Seiten

<a> </a> Mit diesem Tag kannst du Verlinkungen zu anderen seiten erstellen wie in diesem Beispiel gezeigt:
<a href="beliebigeSeite.html">Text Für die Verlinkung<a>. Zum Beispiel: Link auf diese Seite...
<a> <> Mit diesem Befehl kann eine Sprungmarke erstellt werden. Diese ist wie folgt aufzubauen:
<a name="Beispiel">
<a> <> Mit diesem Befehl wird die Sprungmarke angesteuert. Der Befehl wird wie folgt aufgebaut:
<a href="#Beispiel"></a>

Tabellen

<table> </table> Mit diesem Befehl erstellst du eine Tabelle, die folgenden Befehle werden zwischen diese Tags geschrieben.
<tr> <tr> Erstellte eine Zeile.
<td> </td> Zwischen die Tags wird der Text in der Zelle festgelegt.

Medien
<img /> Mit diesem Befehl kannst du ein Bild in die Seite einbinden, dies funktioniert, wie folgt: <img src="[Link zum bild]".
Dann kannst du noch die Höhe und die Breite eingeben. width="BreiteInPixel" height="H&ooml;heInPixel" border="Rahmen" alt="Alternativtext"
title="Titel, der angezeigt wird, wenn man auf das Bild zeigt.". Du kannst sowohl auf andere Bilder auf anderen Webseiten, als auch auf
Bilder auf dem Server/auf dem Pc Verweisen.
<video> </video> Mit diesem Befehl kannst du Videos in deine Webseite einbinden, dies funktioniert, nach diesem Schema
<source src="video.ogg" type="video/ogg" />

Allgemeines

id="" Mit diesem Befehl kannst du den Tags Attribute zuweisen, um sie später besser unterscheiden.
class="" Mit diesem Befehl kannst du Tag in Gruppen ordnen, um der ganzen Gruppe eine Eigenschaft zuweisen zu können.
Relative Links Diese Art von Links verweisen von dem Verzeichnis aus, wo die Seite liegt von der aus verwiesen wird, auf eine seite in einem Unterverzeichnis.
Dies hat den Vorteil, dass man die GANZE Seite verschieben kann, ohne das die Verlinkungen neu gschreiben werden müssen.
Absolute Links Diese Art von Links verweist auf ein Bestimmtes Verzwichnis z.b. "C:\img.jpg". Wenn du deine Seite jedoch auf einen anderen Pc kopierst,
kann die Verzeichnis Strucktur anders sein, dies hat zur Folge, das die Verlinkungen auf deiner Seite nicht mehr stimmen.
